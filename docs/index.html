開始你的健康飲食之旅

EatDiary 是一個全方位的飲食管理應用程式，結合了智能營養追蹤、健康數據管理和社群美食分享功能。無論你是想要控制體重、管理健康飲食，還是尋找美食靈感，EatDiary 都能滿足你的需求。

🌟 主要功能

📸

拍照記錄與分享

  • 快速拍照記錄每一餐
  • 公開分享你的美食照片給其他用戶
  • AI 智能辨識食物（功能開發中）
  • 支援多張照片上傳
  • 探索社群中的美食靈感
📊

營養追蹤

  • 記錄熱量、蛋白質、碳水化合物、脂肪
  • 視覺化營養數據
  • 每日目標設定與追蹤
  • 營養攝取分析報告
  • 飲食習慣洞察
🏃

健康數據完全私密

  • 體重記錄與趨勢圖表
  • 身高管理
  • BMR/TDEE 自動計算
  • 活動量等級設定
  • 個人健康數據僅自己可見
🍽️

食物資料庫

  • 豐富的食物營養資料
  • 常用食物快速新增
  • 自訂食物項目
  • 品牌食品資料庫
  • 快速搜尋功能
🗺️

餐廳地圖

  • 探索附近餐廳
  • 餐廳收藏功能
  • 菜單瀏覽
  • 位置導航
  • 餐廳評價參考
👥

社群功能公開分享

  • 瀏覽其他用戶分享的美食照片
  • 從社群中獲取飲食靈感
  • 發現新的健康食譜
  • 美食探索與交流
  • 建立健康飲食社群
💫

個人化體驗

  • 繁體中文 / English 雙語支援
  • 淺色 / 深色主題
  • 資料雲端同步
  • 跨裝置存取
  • 個人化介面設定

隱私保護

  • 健康數據完全私密（體重、身高等）
  • 食物照片公開分享（可自行選擇是否上傳）
  • 資料加密傳輸
  • 安全的雲端儲存
  • 完整的資料控制權

🔒 您的隱私，我們守護

我們非常重視您的隱私安全。EatDiary 採用分級隱私保護機制，讓您完全掌控自己的資料。

🔵 完全私密資料

體重、身高、BMR/TDEE 等健康數據僅您本人可見，絕不對外公開。

🟠 自主分享資料

食物照片由您決定是否上傳分享，隨時可以刪除。

🔐 加密保護

所有資料透過 HTTPS 加密傳輸，使用安全的雲端儲存系統。

✅ 資料控制權

您隨時可以匯出、刪除或管理您的個人資料。
